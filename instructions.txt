# Python 3.10 Virtual Environment Setup (CPLEX Compatible)

# The .venv is already created with Python 3.10 and CPLEX installed!

# To activate the virtual environment:
# In PowerShell:
.\.venv\Scripts\Activate.ps1

# In Bash/Git Bash:
source .venv/Scripts/activate

# Once activated, simply run:
python sample4.py
